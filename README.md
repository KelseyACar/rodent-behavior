# laboratory-rodent-behavior
This repo houses test data and R scripts to handle raw data from Columbus Instrument's Auto-Track locomotion software (v ) paired with Opto-varimex locomotion equipment, MouseSpecifics Digigait equipment and software, and any manual Rotarod (or other single variable measure) data.
Each behavior type (locomotion, rotarod, gait) has it's own set of functions.
